# Sales Data Analysis using SQL

## 📌 Overview
This project contains SQL queries to analyze sales data, including product performance, trends, and customer behavior.

## 🛠️ Tools
- MySQL

## 📊 Dataset
The dataset includes:
- Product ID
- Product Name
- Quantity
- Price
- Sale Date
- Customer ID

## 🔍 Analysis
- Total sales per product
- Monthly, weekly, yearly trends
- Average, min, max calculations
- Highest and lowest selling products
- Time-based analysis

## 📁 Files
- sales_analysis.sql

## 💡 Skills Used
- GROUP BY
- Aggregate Functions (SUM, AVG, COUNT)
- Date Functions (MONTH, YEAR, WEEK)
- Window Functions
